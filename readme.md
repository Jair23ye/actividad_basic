en esta practica desarrollé una plataforma para la gestion de pedidos y conectar a las personas a que puedan adquirir alimentos saludables
por el momento tenemos la parte funcional  atreves de pruebas CRUD en la tabala de usuarios, restaurantes y menu. Los cuales podra ver al ejecutar el sistema.

1. una vez tengamos el arbol de dependencias de node con "nmp i" podremos proceder a los siguientes pasos. 
2. en el script de BD ya existen usuariso, para acceder al sistema puede usar este correo: prueba@gmail.com y constraseña: 123.
3. debemos levantar el servidor de exprees estamos en el puesto 3000. 
4. una vez ya hayamos levantado el servidor debemos abrir el archivo index.html en nuestro navegador y podremos empezar a interactuar con el sistema
3. Finalmente podemos acceder a probar las funcionalidades de CRUD en las diferentes tablas.
